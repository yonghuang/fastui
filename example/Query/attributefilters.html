查找所有含有 id 属性的 div 元素:div[id]
查找所有 name 属性是 test1 的 input 元素:input[name='test1']
查找所有 name 属性不是 test1 的 input 元素:input[name!='test1']
查找所有 name 以 'test' 开始的 input 元素:input[name^='test']
查找所有 name 以 '1' 结束的 input 元素:input[name$='test']
查找所有 name 包含 'test1' 的 input 元素:input[name*='test']
查找所有含有 id 属性，并且它的 name 属性是以 1 结尾的:input[id][name$='1']
